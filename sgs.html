SGS#

O módulo sgs obtem os dados do webservice do Banco Central, interface json do serviço BCData/SGS - Sistema Gerenciador de Séries Temporais (SGS).

In [1]: from bcb import sgs

In [2]: import matplotlib.pyplot as plt

In [3]: import matplotlib as mpl

In [4]: mpl.style.use('bmh')

In [5]: df = sgs.get({'IPCA': 433}, start='2002-02-01')

In [6]: df.index = df.index.to_period('M')

In [7]: df.head()
Out[7]: 
         IPCA
Date         
2002-02  0.36
2002-03  0.60
2002-04  0.80
2002-05  0.21
2002-06  0.42

In [8]: dfr = df.rolling(12)

In [9]: i12 = dfr.apply(lambda x: (1 + x/100).prod() - 1).dropna() * 100
