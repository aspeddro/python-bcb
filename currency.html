Moedas¶

O pacote tem 2 APIs que dão acesso a informações de moedas.

  • Webscraping no Conversor de Moedas

  • API OData com cotações de taxas de câmbio

Conversor de Moedas¶

O módulo currency obtem dados de moedas do conversor de moedas do Banco Central através de webscraping.

In [1]: from bcb import currency

In [2]: df = currency.get(['USD', 'EUR'], start='2000-01-01', end='2021-01-01', side='ask')

In [3]: df.head()
